# 🤖 Machine Learning Models & Statistical Analysis

**Data-Driven Predictive Modeling, Probability Analysis, and Classification**

This repository highlights my expertise in applied machine learning, focusing on predictive modeling, statistical probability analysis, and algorithmic data classification. It serves as a comprehensive showcase of my ability to take raw, multi-dimensional datasets, process them, and build mathematical models that extract actionable intelligence.

## 🌟 Featured Models & Analyses

### 1. `Heart_Disease_Prediction`
A diagnostic classification model designed to predict heart disease presence based on complex health metrics. Involves rigorous feature scaling, hypothesis testing, and binary classification techniques.

### 2. `Caltrain_Data_Analysis`
A statistical modeling project analyzing Caltrain passenger data. Utilizes probability distributions, expected value calculations, and time-series data parsing to model transportation logistics and boarding rates.

### 3. `Ancestry_Modeling`
A probabilistic model applying Bayes' Theorem and genetic inheritance algorithms to calculate trait probabilities across generational data trees.

### 4. `Logistic_Regression`
Implements a robust Logistic Regression classifier (using the Iris dataset). Features custom mathematical weight extraction, Sigmoid function probability mapping ($$ \sigma(x) = \frac{1}{1 + e^{-x}} $$), and manual cross-validation logic on top of `scikit-learn`.

### 5. `Probability_Analysis`
A suite of Python scripts dedicated to solving complex combinatorial and discrete probability problems (e.g., stochastic simulations, expected values).

## 🧠 Technical Stack
*   **Languages & Libraries:** Python, `scikit-learn`, `pandas`, `numpy`
*   **Concepts:** Binary Classification, Bayesian Inference, Probability Distributions (Poisson, Normal, Binomial), Train/Test splitting, Activation Functions, Stochastic Simulations.
